SENG 440: Embedded Systems

A very inspiring course I am currently taking is SENG 440. The course deals with characterisitcs and design of embedded systems. In this course we will design an application specific integrated circuit (ASIC). I have chosen to work on a project which calculates hyperbolic and trigonometric functions using an algorithm called CORDIC. Fundamentally, it takes a value in radians and outputs the sin, cosine, or tangent using fixed-point arithmetic. Additionally, I will be using the ARM processor to complete this task, and VHSIC Hardware Description Language (VHDL) as a tool for validation.

  • Friendly Arm Website: Friendly Arm
  • CORDIC information: CORDIC Wiki

Brouhaha Android Application Project

Currently, I am working on an Android application named Brouhaha, which takes your current location and provides the user with a random activity to do around them. This application uses Yelp's API to provide users with the activity, rating, distance, and other information that they may find useful. I am using Android Studio to develop the application and currently working on implementing Google Maps to get directions from the user's current location to the activity's address. The application is almost complete, after the Googe Maps is placed, I will work on getting implementing the cost of the activity. I believe this will be the hardest because Yelp's API does not include this information! Though I ran into several debugging problems, I managed to fix them thanks to the abundant Android developers who have ran into the same problems I have. I hope to complete this project by the end of May 2015! I have used several resources to help me get started and are listed below.

  • Android Developer Website: Android Training
  • Yelp's Developer Website: Yelp API
  • Getting started with Scribe (Java OAuth library): Scribe - Java Library
